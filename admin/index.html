Umrechnungsregeln

Klicke auf ⋯ um einen State aus dem ioBroker-Objektbaum zu wählen. Die Ziel-State-ID ist relativ zur Instanz (z.B. converted.temperature).

Aktiv Quell-State Umrechnung Eigene Formel Ziel-State-ID Einheit

Formel-Beispiele: value / 1000  |  Math.round(value * 10) / 10  |  value > 0 ? value : 0

VPD Berechnung (Vapor Pressure Deficit)

Aktiv
Temperatur-State
Luftfeuchtigkeits-State
Ziel-State-ID (relativ)
Formel (Magnus-Approximation):
SVP = 0,6108 · e^(17,27 · T / (T + 237,3)) [kPa]
VPD = SVP · (1 − RH / 100) [kPa]

Eingangswerte: Temperatur in °C, Luftfeuchtigkeit in % (0–100).
Ergebnis erscheint unter changer.<instanz>.<Ziel-State-ID> in kPa.

Richtwerte: < 0,4 kPa = zu feucht  |  0,8–1,2 kPa = optimal  |  > 1,6 kPa = Trockenstress
