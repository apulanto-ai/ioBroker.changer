Umrechnungsregeln

Jede Zeile definiert eine Umrechnung. Die Quell-State-ID muss die vollständige ioBroker-State-ID sein (z.B. hm-rpc.0.ABC.1.TEMPERATURE). Die Ziel-State-ID ist relativ zur Instanz (z.B. converted.temperature).

Aktiv Quell-State-ID Umrechnung Eigene Formel Ziel-State-ID Einheit

Formel-Beispiele: value / 1000  |  Math.round(value * 10) / 10  |  value > 0 ? value : 0
