Umrechnungsregeln

Jede Zeile definiert eine Umrechnung. Quell-State-ID = vollständige ioBroker-ID (z.B. hm-rpc.0.ABC.1.TEMPERATURE). Ziel-State-ID ist relativ zur Instanz (z.B. converted.temperature).

Aktiv Quell-State-ID Umrechnung Eigene Formel Ziel-State-ID Einheit

Formel-Beispiele: value / 1000  |  Math.round(value * 10) / 10  |  value > 0 ? value : 0

VPD Berechnung (Vapor Pressure Deficit)

Aktiv
Temperatur-State-ID
Luftfeuchtigkeits-State-ID
Ziel-State-ID (relativ)
Formel (Magnus-Approximation):
SVP = 0,6108 · e^(17,27 · T / (T + 237,3))  [kPa]
VPD = SVP · (1 − RH / 100)  [kPa]

Eingangswerte: Temperatur in °C, Luftfeuchtigkeit in % (0–100).
Der berechnete VPD-State erscheint unter changer.<instanz>.<Ziel-State-ID> in der Einheit kPa.

Richtwerte: < 0,4 kPa = zu feucht  |  0,8–1,2 kPa = optimal  |  > 1,6 kPa = Trockenstress
